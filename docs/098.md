## 正在写入文件:fputc()函数

fputc()函数用于将单个字符写入文件。它向流中输出一个字符。

**语法:**

```
int fputc(int c, FILE *stream)

```

**示例:**

```
#include main(){
   FILE *fp;
   fp = fopen("file1.txt", "w");//opening file
   fputc('a',fp);//writing single character into file
   fclose(fp);//closing file
} 
```

**file1.txt**

```
a

```
