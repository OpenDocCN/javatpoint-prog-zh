c 程序对每个数字求和:我们只能借助循环和数学运算，用 c 语言编写数字求和程序。

## 数字和算法

要通过 c 程序获得每个数字的总和，请使用以下算法:

*   第一步:按用户获取号码
*   第二步:得到数的模/余数
*   第三步:将剩余的数相加
*   第四步:将数字除以 10
*   第五步:当数字大于 0 时，重复第二步。

让我们看看 c 语言的数字总和程序。

```
#include int main()  
{  
int n,sum=0,m;  
printf("Enter a number:");  
scanf("%d",&n);  
while(n>0)  
{  
m=n%10;  
sum=sum+m;  
n=n/10;  
}  
printf("Sum is=%d",sum);  
return 0;
} 
```

**输出:**

```
Enter a number:654
Sum is=15

Enter a number:123
Sum is=6

```
