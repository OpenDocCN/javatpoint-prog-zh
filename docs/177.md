## 1.星形金字塔图案

### 打印半金字塔的程序

让我们考虑一个打印用于循环的半金字塔图案的例子。

**star.c**

```

#include <stdio.h>
#include <conio.h>
void main()
{
	int i, j, rows;
	printf (" Enter a number to define the rows: \n ");
	scanf("%d", &rows);
	printf("\n");
	for (i = 1; i <= rows; ++i) // outer loop
	{
		for (j = 1; j <= i; ++j) // inner loop
		{
			printf ("* "); // print the Star
		}
		printf ("\n"); 
	}
	getch();	
}

```
