C++ switch 语句从多个条件中执行一条语句。这就像 C++中的 if-else-if 梯形语句。

```
switch(expression){    
case value1:    
 //code to be executed;    
 break;  
case value2:    
 //code to be executed;    
 break;  
......    

default:     
 //code to be executed if all cases are not matched;    
 break;  
}  

```
