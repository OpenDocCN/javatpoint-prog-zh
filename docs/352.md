# c++ deque

> 吴奇珍:t0]https://www . javatppoint . com/post/CPP-deque

Deque 代表双头队列。它概括了队列数据结构，即插入和删除可以从前端或后端执行。

![c++ Deque](img/76f09f2911201661f5b2a25cdfa8c619.png)

## 创建 deque 对象的语法:

```

deque deque_name; 
```
