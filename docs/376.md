## 数字和算法

要通过 C++程序获得每个数字的总和，请使用以下算法:

*   **第一步:**按用户获取号码
*   **第二步:**求数的模/余数
*   **第三步:**将剩余的数相加
*   **第四步:**将数字除以 10
*   **第 5 步:**当数字大于 0 时，重复第 2 步。

让我们看看 C++中的数字总和程序。

```
#include <iostream>
using namespace std;
int main()
{
int n,sum=0,m;  
cout<<"Enter a number: ";  
cin>>n;  
while(n>0)  
{  
m=n%10;  
sum=sum+m;  
n=n/10;  
}  
cout<<"Sum is= "<<sum<<endl;  
return 0;
}

```

输出:

```
Enter a number: 23  
 Sum is= 5

```

