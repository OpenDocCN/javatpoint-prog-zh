# Shell 脚本让命令

> 原文:[https://www.javatpoint.com/shell-let-scripting](https://www.javatpoint.com/shell-let-scripting)

**让**命令是一个算术运算符。几乎和(( ))一样。唯一不同的是，let 是一个算术命令，而(( ))是一个复合命令。

它是一个内置命令，指示 shell 执行算术表达式的计算。使用**让**命令的算术运算符周围不应有空格。
